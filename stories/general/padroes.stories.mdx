# Arquitetura

Uma das atribuições do time **TheOffice** é **padronizar a arquitetura global CSS** das aplicações. Como a iniciativa começou **após**
as aplicações já estarem implementadas, **as novas** features implementadas devem seguir o **padrão criado** a seguir. O **legado** continuará
existindo e será **eventualmete migrado**.
