# Tokens

**[Tokens no figma](https://www.figma.com/file/4QvnRfhttlJ2hIKQbuEPZl/Tokens?node-id=308%3A120)**

Os tokens sao formas de padronizar a identidade visual de componentes e layout tanto no processo de criação, como de
diagramação.

Para utilizar os tokens no código, basta utilizar o token presente no figma, e utilá-los como CSS properties.
