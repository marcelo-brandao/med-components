# Tokens

Os tokens são uma forma de **Facilitar** e **Padronizar** a identidade visual dos produtos tanto no processo de **Criação** quanto no de **Desenvolvimento**.
Com isso os processos se dão com **Qualidade** e **Agilidade**, promovendo **Padronização** e **Reusabilidade**, minimizando **Custos** e **Retrabalho**.

## Utilização

Os tokens estão presentes em:

- **[Tokens de Cores](http://desenv.ordomederi.com/med-components/?path=/story/tokens-cores--cores)**
- **[Tokens de Tipografia](http://desenv.ordomederi.com/med-components/?path=/story/tokens-tipografia--tipografia)**
- **[Tokens de Espaçamento](http://desenv.ordomederi.com/med-components/?path=/story/tokens-espacamento--espacamento)**
- **[Tokens Complementares](http://desenv.ordomederi.com/med-components/?path=/story/tokens-complementares--complementares)**

Para utilizá-los basta referência-los como **CSS Properties** no código.

## ⚠️ IMPORTANTE!!!

- Para utilizar o token **$med-line-height-default** em **[Tipografia](http://desenv.ordomederi.com/med-components/?path=/story/tokens-tipografia--tipografia)** é necessário o uso da função SASS **[line-height()](http://desenv.ordomederi.com/med-components/?path=/story/global-functions-tipografia--page#line-height)**.
- Para utilizar os tokens **$med-letter-spacing-*** em **[Tipografia](http://desenv.ordomederi.com/med-components/?path=/story/tokens-tipografia--tipografia)** é necessário o uso da função SASS **[letter-spacing()](http://desenv.ordomederi.com/med-components/?path=/story/global-functions-tipografia--page#letter-spacing)**.

## Exemplo

**⚠️ IMPORTANTE!!! Exemplificação do uso dos tokens. Os componentes não devem ser criados dessa maneira...**

Nesse exemplo vamos utilizar os tokens presentes na seguinte **[seção](http://desenv.ordomederi.com/med-components/?path=/story/tokens-exemplo--exemplo)**.

### Markup

```html
<div class="nav-bar">
  <img class="icon icon--left" src="icon.svg" alt="icon">
  <h1 class="nav-bar__heading">page title</h1>
  <img class="icon icon--right" src="icon.svg" alt="icon">
</div>
```

### Estilização

```scss
.nav-bar {
  display: flex;
  align-items: center;
  justify-content: space-between;

  padding: var(--med-spacing-squish-xs);
  background: var(--med-color-neutral-dark-prime);

  &__heading {
    text-transform: uppercase;

    font-size: var(--med-font-size-xs);
    padding-top: var(--med-spacing-squish-xs);
    color: var(--med-color-neutral-light-prime);
    letter-spacing: var(--med-line-height-compressed);
  }
}

.icon {
  min-width: 24px;
  min-height: 24px;

  &--left {
    padding-right: var(--med-spacing-inline-base);
  }

  &--right {
    padding-left: var(--med-spacing-inline-base);
  }
}

```
