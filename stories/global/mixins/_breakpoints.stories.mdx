# Breakpoints

Mixins **globais** relacionados a breakpoints.

## Utilização

No arquivo que a utilização dos mixins sejam necessárias, importar:

``` scss
@import '~@mbrandao/med-components/src/global/global';
```

## O que são?

São as quebras de layout padrões adotadas junto ao time de designers.
Ao todo temos 6 quebras que vão do menor ao maior possível, bem como um limitador máximo para que não precisemos
nos preocupar com resoluções acima do fullHD por enquanto, pois não é o propósito do app.

<img src={breakpoints} alt="breakpoints" />

## breakpoint-up()

### Utilização:

```scss
.class {
  @include breakpoint-up('md') { ... }
}
```

### Compila para:

```scss
@media (min-width: 768px) {
  .class { ... }
}
```

## breakpoint-down()

### Utilização:

```scss
.class {
  @include breakpoint-down('lg') { ... }
}
```

### Compila para:

```scss
@media (max-width: 991.98px) {
  .class { ... }
}
```

## breakpoint-between()

### Utilização:

```scss
.class {
  @include breakpoint-between('sm','lg') { ... }
}
```

### Compila para:

```scss
@media (min-width: 576px) and (max-width: 991.98px) {
  .class { ... }
}
```

## breakpoint-only()

### Utilização:

```scss
.class {
  @include breakpoint-only('xxs') { ... }
}
```

### Compila para:

```scss
@media (min-width: 576px) and (max-width: 767.98px) {
  .class { ... }
}
```
