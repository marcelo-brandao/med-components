# Variables

**Variáveis SCSS globais** necessárias para o uso junto as **functions** e **mixins SCSS**.

| **$med-font-size-sm**   | Define o valor do token **$med-font-size** para **sm**   | Number | **20px** |
| **$med-font-size-md**   | Define o valor do token **$med-font-size** para **md**   | Number | **24px** |
| **$med-font-size-lg**   | Define o valor do token **$med-font-size** para **lg**   | Number | **32px** |
| **$med-font-size-xl**   | Define o valor do token **$med-font-size** para **xl**   | Number | **40px** |
