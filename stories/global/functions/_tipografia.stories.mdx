# Tipografia

## O que são?

São funções que foram criadas para facilitar o uso das propriedades **line-height** e **letter-spacing**, pois estas necessitam
de um calculo para serem utilizadas. É necessária a utilização dos **[tokens]()** junto a elas.

## line-height

Função criada pois o valor do line-height necessita do cálculo **(ceiling((font-size \* 1,3) 8))** para ser aplicado.

```scss
@function line-height($font-size: $med-font-size-xs) {
  $line-height: ceil(strip-unit($font-size) * 1.3);

  @while ($line-height % 8) != 0 {
    $line-height: $line-height + 1;
  }

  @return #{$line-height}px;
}
```

**Exemplo:**

```
:host {
  .class {
    line-height: line-height($med-font-size-md)
  }
}
```

## letter-spacing

Função criada pois o valor do letter-spacing leva em conta o tamanho da fonte sendo utilizada para ser calculado.

```scss
@function letter-spacing(
  $font-size: $med-font-size-xs,
  $letter-spacing: $med-line-spacing-default
) {
  @return $font-size * $letter-spacing;
}
```

**Exemplo:**

```
:host {
  .class {
    letter-spacing: letter-spacing($med-font-size-md, $med-line-spacing-default);
  }
}
```
