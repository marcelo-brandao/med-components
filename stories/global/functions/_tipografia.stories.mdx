# Tipografia

Funções que foram criadas para facilitar o uso das propriedades CSS **line-height** e **letter-spacing**, pois estas necessitam
de um cálculo para serem utilizadas. É necessária a utilização dos tokens **[$med-font-size](http://desenv.ordomederi.com/med-components/?path=/story/global-variables--page#tokens-med-font-size) e [$med-letter-spacing](http://desenv.ordomederi.com/med-components/?path=/story/global-variables--page#tokens-med-letter-spacing) no formato SCSS** junto a elas.

## line-height()

O valor da propriedade CSS **line-height** necessita do cálculo **(ceiling((font-size \* 1,3) 8))** para ser gerado.

### Utilização:

```scss
.class {
  line-height: line-height($med-font-size-md);
}
```

### fn: line-height($font-size: $med-font-size-xs);

| Parâmetro      | Descrição                                                                                                                                                                                                         | Tipo   | Default               |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ | --------------------- |
| **$font-size** | Valor dos tokens **[$med-font-size](http://desenv.ordomederi.com/med-components/?path=/story/global-variables--page#tokens-med-font-size)** necessário para geração do valor final da propriedade **line-height** | Number | **$med-font-size-xs** |

## letter-spacing()

O valor da propriedade CSS **letter-spacing** leva em conta o tamanho da fonte sendo utilizada no elemento para ser calculado.

### Utilização:

```scss
.class {
  letter-spacing: letter-spacing($med-font-size-md, $med-line-spacing-default);
}
```

### fn: letter-spacing($font-size: $med-font-size-xs, $letter-spacing: $med-line-spacing-default);

| Parâmetro           | Descrição                                                                                                                                                                                                                          | Tipo   | Default                       |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ | ----------------------------- |
| **$font-size**      | Valor dos tokens **[$med-font-size](http://desenv.ordomederi.com/med-components/?path=/story/global-variables--page#tokens-med-font-size)** necessário para geração do valor final da propriedade CSS **letter-spacing**           | Number | **$med-font-size-xs**         |
| **$letter-spacing** | Valor dos tokens **[$med-letter-spacing](http://desenv.ordomederi.com/med-components/?path=/story/global-variables--page#tokens-med-letter-spacing)** necessário para geração do valor final da propriedade CSS **letter-spacing** | Number | **$med-line-spacing-default** |
