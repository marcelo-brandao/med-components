# Variables

**Variáveis SASS** utilizadas **apenas** na aplicação **medsoft**.

## Utilização

No arquivo em que a utilização das **variáveis** forem **necessárias**, importar:

``` scss
@import '~@mbrandao/med-components/src/medsoft/medsoft';
```

| Variable            | Descrição                                                                                                                                                  | Tipo   | Default  |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ | -------- |
| **$med-iphone-x-top** | Define a distância para alocar um elemento referente ao **[chanfrado](https://support.apple.com/en-ca/HT207354)** dos dispositivos da família iphone X   | Number | **40px** |
| **$med-iphone-top**   | Define a distância para alocar um elemento referente aos **[ícones de status](https://support.apple.com/en-ca/HT207354)** dos dispositivos iphone padrão | Number | **20px** |
| **$med-mac-top**      | Define a distância para alocar um elemento referente aos botões de controle de uma aplicação desktop macOS                                               | Number | **15px** |
